# 占位符

占位符可以用于以下图标属性：

- `ACTIONS`
- `NAME`
- `LORE`
- `SKULL-OWNER`

占位符标:

| 占位符 | 描述 |
| ----------- | ----------- |
| {player} | 玩家名 |
| {money} | 玩家的钱 |
| {world} | 玩家所处世界 |
| {online} | 服务器在线人数 |
| {max_players} | 最大在线人数 |

使用 PlaceHolderAPI 提供的占位符

**Note**:必须要带百分号 如 %player_gamemode%
