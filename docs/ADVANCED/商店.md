# 商店

通过结合一些[物品属性](/docs/BASICS/物品属性.md)和[动作](/docs/BASICS/点击动作.md)，你可以创建简单的商店，出售和购买玩家的物品。

需要有效的 [经济](/docs/BASICS/安装.md) 设置

## 出售
```
example:
    PRICE: 100
    ACTIONS:
      - 'give: diamond, 1'

    NAME: 'Buy a diamond'
    MATERIAL: diamond
    POSITION-X: 1
    POSITION-Y: 1
```
## 购买
```
example:
    REQUIRED-ITEMS:
      - 'dirt, 64'
    ACTIONS:
      - 'give-money: 10'

    NAME: 'Sell dirt'
    MATERIAL: dirt
    POSITION-X: 1
    POSITION-Y: 1
```
